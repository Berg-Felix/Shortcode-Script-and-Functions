Adicione Scripts, Shortcode e funcionalidades em suas páginas web.

== Description ==
O plugin Shortcode Script and Functions é composto por uma tab de três páginas onde a primeira tab contém a inserção de scripts no Header, Footer ou em uma página especifica.
A segunda  tab contém  a  adição de shortcode estático e shortcode de link. 
A Terceira tab contém algumas funcionalidades básica para complementar suas páginas web


== Installation ==
1.Faça o upload da pasta 'Shortcode Script and Functions' para o 'plugins / / wp-content /' diretório.
2.Ative o plugin através do menu 'Plugins' no WordPress.
3.Clique na página de opções do 'Shortcode Script and Functions'.
